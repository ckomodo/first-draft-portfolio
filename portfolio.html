Ven-U

VenU is a music-discovery website for fans of live music discover new artists and shows. The search functionality allows you to search for a band and retrieve a live performance only by said band. There is also currently live broadcasts for the user to browse, when available.

This was my first real-world application to make. Other than the overall design, my other roles included ensuring the nav-bar is responsive, setting up all the cards and getting 24/7 live-streaming channels. I learned to use Materialize, which is a CSS framework I had not used before. I learned how in coding, team work achieves so much in lesser time than coding solo and to be careful with Git!
